# Dijital Alışkanlık Analiz Sistemi

Bu proje, kullanıcıların ekran sürelerini ve dijital alışkanlıklarını analiz etmeyi sağlayan web tabanlı bir dashboard uygulamasıdır.

## 🛠️ Kullanılacak Teknolojiler
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Backend:** Java, Spring Boot
* **Veritabanı:** MySQL
* **Geliştirme Ortamı:** Visual Studio Code

## 🚀 Geliştirme Ortamı Kurulum Adımları
Tüm takım üyelerinin projeyi kendi yerel bilgisayarında (local environment) çalıştırabilmesi için aşağıdaki adımlar izlenmelidir:

1. **Repoyu Klonlama:**
   Terminal üzerinden `git clone https://github.com/Elldiana/dijital_aliskanlik_analiz_sistemi.git` komutu çalıştırılmalıdır.
2. **Geliştirme Ortamını Ayarlama:**
   Klonlanan proje klasörü VS Code ile açılmalıdır.
